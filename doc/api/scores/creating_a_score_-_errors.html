Scores API

Create, read & delete scores

Creating a score - errors

POST /api/scores

Parameters

Name Description
score[name] Name of player
score[score_point] Score of player
score[score_at] Score time of player

Request

Headers

Content-Type: application/json

Route

POST /api/scores

Body

{
  "score": {
    "name": null,
    "score_point": "score",
    "score_at": "2021-09-20 12:00:00 JST"
  }
}

cURL

curl "http://localhost:3000/api/scores" -d '{
  "score": {
    "name": null,
    "score_point": "score",
    "score_at": "2021-09-20 12:00:00 JST"
  }
}' -X POST \
	-H "Accept: application/json" \
	-H "Content-Type: application/json"

Response

Headers

Content-Type: application/json; charset=utf-8

Status

400 Bad Request

Body

{
  "errors": [
    {
      "code": 400001,
      "message": "can't be blank",
      "field": "name"
    },
    {
      "code": 400003,
      "message": "is not a number",
      "field": "score_point"
    },
    {
      "code": 400024,
      "message": "is invalid",
      "field": "score_at"
    }
  ]
}
